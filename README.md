## 🚀 Featured Projects

### StudyForge Study App

StudyForge is a front end study application that allows users to search study topics, generate quizzes, take quizzes, and review saved results. The project uses live API data, dynamic quiz generation, localStorage, and pass/fail progress tracking.

**Skills used:** JavaScript, HTML, CSS, jQuery, Wikipedia API, Open Trivia DB API, dynamic rendering, localStorage, and user focused interface design.

**Project Highlights:**

- Searches live API data
- Dynamically generates quizzes
- Saves completed results with localStorage
- Displays pass/fail progress for review
- Includes image carousel controls
- Includes a functional jQuery accordion for study tips

**Links:**

- Live Demo: https://jonnunez03.github.io/studyforge/
- GitHub Repo: https://github.com/jonnunez03/studyforge

### Poke Picker Web App

Poke Picker is a Pokémon themed interactive site with a randomized Pokémon carousel. The site includes a Pokémon guessing game where users enter a number to try and catch a Pokémon. It also includes a live Pokémon card, features page, about page, working contact form with validation, and light/dark mode.

**Skills used:** API data retrieval, JavaScript logic, DOM manipulation, application state management, form validation, responsive layout testing, and user focused interactions.

**Project Highlights:**

- Uses PokéAPI for Pokémon data and artwork
- Allows users to select Pokémon from a carousel
- Includes a guessing game to catch a Pokémon
- Displays a Pokémon card after a successful guess
- Includes form validation with JavaScript and regular expressions
- Includes light/dark mode

**Links:**

- Live Demo: https://jonnunez03.github.io/poke-picker-web-app/
- GitHub Repo: https://github.com/jonnunez03/poke-picker-web-app

### Lord of the Forms

Lord of the Forms is a React and TypeScript form application with controlled inputs, live validation feedback, error messages, and submitted information display. The form validates user information and only clears error messages when the user actively types valid input.

**Skills used:** React, TypeScript, controlled inputs, form validation, state management, error handling, and user feedback.

**Project Highlights:**

- Built with React and TypeScript
- Uses controlled form inputs
- Displays live validation feedback
- Shows error messages until valid input is entered
- Displays submitted user information after completion
